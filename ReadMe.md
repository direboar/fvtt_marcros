# FVTT用のマクロ作成用リポジトリ
* [このリポジトリ](https://github.com/foundry-vtt-community/macros/tree/master/5e)にあるマクロを、使いやすいように一部カスタマイズ。
* 自分が使いやすいものを追加。

# 使い方
* srcから、使いたいマクロを選ぶ
* FVTT上でマクロを作成。
* Typeをscriptに変更し、使いたいマクロをコピーペースとして保存する。
