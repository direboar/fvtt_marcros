# FVTT用のマクロ作成用リポジトリ
* 以下のリポジトリにあるマクロを、使いやすいように一部カスタマイズ。
  https://github.com/foundry-vtt-community/macros/tree/master/5eの内容をいったん持ってきた状態
* 自分たちが使いやすいものを追加。

# 使い方
* srcから、使いたいマクロを選ぶ
* FVTT上でマクロを作成。
* Typeをscriptに変更し、使いたいマクロをコピーペースとして保存する。
